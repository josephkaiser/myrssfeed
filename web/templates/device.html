Trust this site on your device

myRSSfeed uses a private certificate that your browser doesn't recognise yet. Follow the two steps below to stop the security warning.

Download the certificate
This small file tells your device to trust connections to myRSSfeed. It only applies to this local network — it does not affect any other website.
⇓ Download certificate
Install it on your device
Instructions vary by operating system — pick yours:

Easiest: tap the button below in Safari — iOS will guide you through the install in one step.

⇓ Install profile (Safari)
  1. After downloading, go to Settings and tap Profile Downloaded near the top.
  2. Tap Install (top right), enter your passcode, tap Install again.
  3. Go to Settings › General › About › Certificate Trust Settings.
  4. Toggle on myRSSfeed CA under "Enable Full Trust for Root Certificates".
  5. Open Safari and go to https://myrssfeed.local.

Must use Safari to download the profile — Chrome and Firefox on iOS won't trigger the install prompt.

  1. Tap Download certificate above — the file saves to Downloads.
  2. Open Settings › Security › Encryption & credentials › Install a certificate.
  3. Choose CA certificate, then select the downloaded file.
  4. Confirm the warning and tap Install anyway.
  5. Open Chrome and go to https://myrssfeed.local.

Exact steps vary by Android version. Search "install CA certificate" + your phone model if needed.

Easiest — automatic install: download bootstrap.command from GitHub and double-click it. It fetches the certificate and installs it in one step.

⇓ Download bootstrap.command

Or use the Apple profile (same as iOS — installs via System Preferences): Download profile

Manual steps instead
  1. Click Download certificate above.
  2. Double-click the downloaded myrssfeed-ca.crt — Keychain Access opens.
  3. Double-click the myrssfeed certificate in the list.
  4. Expand Trust, set When using this certificate to Always Trust.
  5. Close and enter your password to confirm.
  1. Click Download certificate above.
  2. Double-click the downloaded file and click Install Certificate…
  3. Choose Local Machine and click Next.
  4. Select Place all certificates in the following store → Browse → Trusted Root Certification Authorities.
  5. Click Next → Finish → confirm the security prompt.
  6. Reload https://myrssfeed.local.

Firefox has its own certificate store — also import via Firefox Settings › Privacy & Security › Certificates › View Certificates › Import.

  1. Click Download certificate above.
  2. Open Firefox Settings › Privacy & Security › Certificates › View Certificates.
  3. On the Authorities tab click Import, select the file.
  4. Check Trust this CA to identify websites and click OK.
  5. Reload https://myrssfeed.local.

For Chrome on Linux:
sudo cp myrssfeed-ca.crt /usr/local/share/ca-certificates/ && sudo update-ca-certificates
